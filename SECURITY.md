# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in our project, please let us know by emailing us at [security@textile.io](mailto:security@textile.io). We take security issues very seriously and will respond to your report as quickly as possible. Please do not publicly disclose the issue until we have addressed it.

## Supported Versions

We are committed to maintaining the security of our project, and we will provide security updates for the latest stable release as well as the previous stable release. If you are using an older version of our project, we recommend upgrading to the latest stable release.

## Vulnerability Disclosure Process

Once we receive your report, we will follow the following steps:

1. We will acknowledge receipt of your report within 24 hours.
2. Our security team will investigate the issue and determine the scope and impact of the vulnerability.
3. We will work on a fix for the vulnerability and deploy it as soon as possible.
4. Once the fix is deployed, we will notify you and publicly disclose the vulnerability.

## Bug Bounty Program

We do not currently have a bug bounty program, but we appreciate any help identifying security vulnerabilities in our project. If you discover a vulnerability, please follow the reporting process outlined above.
